Application navigation:
	Appointments main screen, customers screen, and reports screen are selected
	from the "View" menu of the Main screen. Selecting "Appointments For The Week/Month"
	radio buttons alternates between calendar views. Selecting "Appointments" from "View" menu
	displays all appointments in DB. "File" menu in Main screen is used to logout or 
	close the application.
